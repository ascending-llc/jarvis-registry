This is the initial tracked release of Jarvis Registry. Previous versions
(asc0.1.x through asc0.2.11) were pre-release builds without formal release notes.

## What's included in asc0.2.12

✨ Features
- MCP/Agent gateway with OAuth2 authentication via Keycloak and AWS Cognito
- Semantic search over registered MCP servers and A2A agents at runtime
- Unified access control with RBAC scopes across all registered tools
- Docker Compose full-stack deployment profile
- REST API for dynamic tool discovery and agent registration

🌍 Documentation
- Full documentation available at jarvisregistry.com
- Quick-start guide, deployment runbook, and RBAC reference
